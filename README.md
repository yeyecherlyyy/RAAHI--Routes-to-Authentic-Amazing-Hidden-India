# 🌍 RAAHI – Routes to Authentic & Amazing Hidden India  

RAAHI is a *community-driven travel platform* that helps users discover *hidden gems across India* that are often overlooked by mainstream travel apps. Instead of focusing on popular tourist destinations, RAAHI highlights *authentic local experiences* shared by locals.

The platform combines *interactive maps, AI-powered recommendations, local guides, and community features* to create a more meaningful and personalized travel experience.

---

## 🚀 Features  

### 🗺️ Hidden Gem Discovery  
- Interactive map with *custom pins*  
- Filter by categories (food, culture, markets, etc.)  
- Detailed place pages with photos, stories, and reviews  

### 🤖 AI-Powered Recommendations (Gem Genie)  
- Natural language search for discovering places  
- Personalized suggestions based on user preferences  

### 👤 Tourist Portal  
- User profiles with travel history, reviews, badges, and points  
- Personalized exploration and itinerary planning  

### 🧭 Guide / Local Portal  
- Verified local guides and agencies  
- Profiles with expertise and itineraries  
- Booking system for guided experiences  

### 💳 Booking & Payments  
- Secure booking system  
- Partial advance payment via Razorpay  

### 🧑‍🤝‍🧑 Community Features  
- Connect with like-minded travelers  
- Find travel buddies  
- Share experiences and reviews  

### 🏆 Gamification  
- Earn badges, points, and rewards  
- Leaderboards and engagement system  

---

## 🛠️ Tech Stack  

- *Frontend:* Next.js, TypeScript, Tailwind CSS  
- *Backend:* Node.js / Next.js API Routes  
- *Database:* Supabase (PostgreSQL)  
- *Maps:* Mapbox API  
- *Image Storage:* Cloudinary  
- *AI:* Claude API (Gem Genie)  
- *Payments:* Razorpay  

---

## 🔄 How It Works  

1. User signs up and creates a profile  
2. Sets location or searches a city  
3. Explores hidden gems on an interactive map  
4. Gets recommendations (filters + AI)  
5. Connects with travel buddies or selects a guide  
6. Books a guide and experiences the trip  
7. Shares reviews and contributes new gems  

---

## 🎯 Problem It Solves  

- Travel apps focus on *crowded tourist spots*  
- Hidden gems remain *undiscovered*  
- Lack of *personalization and local connection*  

👉 RAAHI bridges this gap using *local insights + community + technology*

---

## 🌟 Vision
